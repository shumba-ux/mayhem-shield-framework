# Methodology

Mayhem Shield uses an implementation-assurance methodology for enterprise AI deployments.

The question is not whether a vendor says a control exists. The question is whether that control actually works for the specific deployment being reviewed.

## Review model
Each review is built from:
- the core framework
- one primary implementation category
- all applicable overlays

## What the methodology evaluates
- architecture and trust boundaries
- identities and access paths
- data handling and retention boundaries
- connector and integration exposure
- workflow execution and approval conditions
- evidence needed for go-live decisions
