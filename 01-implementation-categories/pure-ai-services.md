# Pure AI Services

## Definition
Direct model or API services used by applications, workflows, or developers.

## Typical examples
Model APIs, direct inference services, managed model endpoints.

## What makes this category distinct
The main control boundary is the direct API/service relationship rather than a vendor-managed SaaS workspace.

## Common overlay pairings
- RAG
- Agentic Workflow Execution
- Self-Hosted / Private Deployment
- Regulated Data
